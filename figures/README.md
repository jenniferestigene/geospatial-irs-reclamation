# Figures Directory

This directory contains visualizations and outputs from the geospatial analysis.

- **E14-GeoAnalysis_Poster_EstigeneLanceta.pdf** - Academic poster presenting the full analysis

When you run the R Markdown analysis, additional figures will be generated and saved here:

### Statistical Graphics
- Cross-K function plots
- Distance distribution charts
- Project type compositions
- Temporal comparison visualizations

## File Formats

- **PDF:** High-resolution academic outputs
- **PNG:** Web-friendly images for documentation
- **HTML:** Interactive maps and widgets
