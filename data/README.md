# Data Directory

This directory contains the spatial datasets used in the analysis:

## Data Sources

1. **Former IRS Locations**
   - Source: Borealis Data Repository
   - DOI: 10.5683/SP2/FJG5TG
   - Description: Officially recognized former Indian Residential School sites across Canada

2. **Indigenous Infrastructure Projects (IIP)**
   - Source: Indigenous Services Canada
   - URL: geo.sac-isc.gc.ca
   - Description: Database of Indigenous-led infrastructure development projects

3. **Provincial Boundaries**
   - Source: Statistics Canada
   - URL: https://www12.statcan.gc.ca/census-recensement/alternative_alternatif.cfm
   - Description: Canadian provincial and territorial geographic boundaries

## Data Usage Notes

- All spatial data is projected to EPSG:3347 (Statistics Canada Lambert) for analysis
- Missing or incomplete coordinates have been geocoded using place names
- Project classifications are based on Indigenous Services Canada categories

## Privacy and Ethics

This data involves sensitive information about Indigenous communities and colonial history. Users should:
- Respect Indigenous data sovereignty principles
- Use data only for research and educational purposes
- Acknowledge sources appropriately
- Consider community impacts when sharing results
