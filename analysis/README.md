# Analysis Directory

This directory contains generated analytical outputs from the geospatial analysis.

## Expected Outputs

When you run the R Markdown analysis, various CSV files, maps, and visualizations will be saved here.

### File Types

- **CSV files:** Statistical summaries and data tables
- **PNG/PDF files:** Maps and visualizations  
- **HTML files:** Interactive maps

## How to Generate

Run the R Markdown notebook to generate outputs:

```r
rmarkdown::render("notebook/E14-GeoAnalysis_Poster-Code_EstigeneLanceta.Rmd")
```
